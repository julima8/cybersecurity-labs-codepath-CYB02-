# Week 1 – Network Traffic Analysis (Wireshark)

## Lab: Rogue User Investigation

### Objective
Analyze network traffic and logs to identify suspicious activity and determine if a user was impersonated.

### Tools Used
- Wireshark
- DHCP Logs
- Security Logs

### What I Did
- Analyzed a `.pcap` file using Wireshark  
- Applied filters (SMTP) to isolate relevant traffic  
- Identified the source IP address linked to suspicious email activity  
- Correlated IP with DHCP logs to identify the host machine  
- Used security logs to determine which user was logged in

## Project: Email Traffic Analysis

### Objective
Differentiate between legitimate and fraudulent email activity.

### What I Did
- Analyzed email traffic patterns to identify anomalies  
- Evaluated sender behavior and message characteristics  
- Applied knowledge of SMTP protocol to assess legitimacy  
- Identified indicators of suspicious or fraudulent email activity
